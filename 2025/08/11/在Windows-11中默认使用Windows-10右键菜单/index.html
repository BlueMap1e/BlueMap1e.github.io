在Windows 11中默认使用Windows 10右键菜单

Posted on 2025-08-11 Edited on 2025-08-29 In Tech , Windows Views: Disqus:
Symbols count in article: 502 Reading time ≈ 3 mins.

前言

最近重装了系统，觉得Windows 11默认的右键菜单方案很别扭，还是Windows 10的样式最舒适。因此找了找网上的方案，最后自己摸索出可用的方法。并不困难。

修改注册表操作可能存在一定风险，如果担心风险，可以考虑备份，或者放弃修改。

如果不想永久性修改，按下Shift+右键即可，可以呼出Windows 10风格的详细右键菜单。

原理简介

（信息来源于Google Gemini）

微软在 Windows 11 中为文件资源管理器（Explorer）引入了新的上下文菜单（右键菜单）机制。它通过一个被称为 CLSID（Class ID）的特殊标识符来加载。{86ca1aa0-34aa-4e8b-a509-50c905bae2a2} 就是用来控制这个新菜单的关键 ID。

通过在注册表中为这个 ID 增加一个 InprocServer32 项并设置其为空值，我们实际上是在告诉系统：“不要加载与此 CLSID 关联的默认处理程序，请退回到旧版（经典版）的上下文菜单。” 这样就巧妙地绕过了 Windows 11 的新设计，让经典的 Windows 10 菜单得以重现。

结语

本文简单介绍了如何在Windows 11操作系统中用简单的注册表修改手段将右键菜单样式退回经典的Windows 10。
