Milovana EOS Editor简化教程 ——如何简单上手制作一个Webtease

Posted on 2024-09-19

引言

（这篇教程正在施工和逐渐完善）

EOS Editor是Milovana的一个Web UI，可以用于制作简单的互动小游戏。

Milovana EOS Editor

一个典型且成功的中文作品是鹭鸣有音制作的《鹭音大人的自慰挑战》。

（注：这个作品包含的倾向包括恋袜癖，恋足癖，抖M，Sub，辱骂，寸止，污言等，尝试前最好先注意一下）

鹭音大人的自慰挑战 (milovana.com)

注册

要制作Milovana Webtease，你需要注册一个Milovana账号。

这个平台注册是邀请制的，请注意。但不用担心，这个门槛仅仅是一些等待时间而已。

Milovana Editor

你需要一个邀请码，为了获得这个邀请码，需要进入一个subReddit的帖子：

Invite code thread : r/Milovana (reddit.com)

然后你只需要在回帖中发一个礼貌的请求就可以了，例如这样：

过一段时间，Milovana的工作人员会给你回帖”Sent.”，并给你私信一个邀请码。

使用这个邀请码就可以注册Milovana账号，注册后就可以使用，制作Webtease。
