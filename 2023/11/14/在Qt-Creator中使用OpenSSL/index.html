# 前言

最近在折腾 Qt，刚好可以做一些加解密的桌面应用来玩。因此就尝试在 Qt Creator 中集成 OpenSSL。

# 环境

OS: Windows 10/11

IDE: Qt Creator 11.0.3 (Open Source) 下载链接：Download Qt OSS: Get Qt Online Installer

OpenSSL: OpenSSL 3.1.2 Win64 下载链接：Win32/Win64 OpenSSL Installer for Windows - Shining Light Productions (slproweb.com)

# QMake

QMake 集成 OpenSSL 非常简单。

新建 Qt 项目时选择 QMake，然后进入项目的 .pro 文件，添加这两行：

（PS：把示例中的路径修改为你自己的 OpenSSL 安装路径）

INCLUDEPATH += D:\Lib\OpenSSL-Win64\include
LIBS += -L"D:\Lib\OpenSSL-Win64\lib" -lssl -lcrypto
