简单的Cloudflare Warp使用方法

# 前言

Cloudflare Warp 是 Cloudfare 推出的一种免费、高性能的 VPN 服务，理论上能够提升上网的安全性与性能，也能用来绕过网络审查。

本文字教程主要介绍在个人设备上使用 Cloudflare Warp 连接到互联网，并通过 Cloudflare Zero Trust 登录客户端，解除 Warp 自身的流量限制。

# 安装与连接

安装非常简单，在 Cloudflare 1.1.1.1 的官方网站下载适合自己设备的客户端即可。

下载链接：1.1.1.1 — The free app that makes your Internet faster.

打开安装包，按照指示完成安装即可。打开 Cloudflare Warp 客户端。

点击连接，即可尝试连接。这一步有一定的可能性会失败。如果成功，已经可以访问国际互联网，而且性能通常不错。但是有一定的流量限制。

如果是轻度使用者，不在意流量限制，到这一步即可完成。

# 注册 Zero Trust

登录 Cloudflare Dashboard：https://dash.cloudflare.com/

当然，如果没有账号可以先注册一个，很容易。注册过程略。

在左侧找到 Zero Trust，注册一个 Zero Trust 账号，团队名称可以自由选择。注册过程略。

注册成功后，找到左侧的 My Team 一栏，找到 Devices 选项。选择 Connect a Device 选项。

单击 Create an Enrollment ，输入需要注册的邮箱后缀名，例如 @gmail.com ，然后点击 save 。

# Zero Trust 登录与连接

打开自己的 Cloudflare Warp 客户端，在右键菜单中找到 偏好设置 ，找到 账户 一栏，点击 使用Cloudflare Zero Trust登录 。

输入自己之前注册 Zero Trust 使用的团队名称，点击确认，会跳转到团队的一个 Web 验证界面。

输入自己使用的邮箱。

在自己的邮箱当中找到 Cloudflare 发送的验证码，填写验证码，确认。

稍作等待，Cloudflare Warp 客户端的 Zero Trust 将成功登陆。

打开 Cloudflare Warp 客户端，现在 Warp 的标志将被替换为 Zero Trust。点击连接按钮，即可尝试连接到 Cloudflare Warp 服务。

这一步的成功率很高，通常能够连接成功。并且 1G 的流量限制已经解除。

现在可以尝试访问一些网站，例如 Google。

也可以尝试进行网络测速。

# 结语

至此，Cloudflare Warp + Zero Trust 的简单使用方法已经介绍完毕。Warp 能够通过 WireGuard 协议保护互联网协议，拥有强大的安全性、高效的协议设计和很高的下行速率（有一个问题是上行速率较低），并通过 1.1.1.1 DNS 解析来提高 DNS 解析的效率，避免 DNS 污染的影响，改善综合上网体验。
